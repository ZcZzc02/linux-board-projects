# LoRa E22 接线与 UART 选择

当前项目以交接文档中已经验证过的接线为准，不建议重新拔线。

## 已确认 UART

K7 侧使用：

```text
/dev/ttyS3
```

运行时设备树期望：

```text
uart3 status: [okay]
i2c7 status: [disabled]
```

## E22 配置模式

读取 E22 配置时使用配置模式：

- `M0 -> GND`
- `M1 -> 3.3V`

串口参数：

```text
9600 8N1
```

读配置命令：

```text
C1 00 09
```

期望响应：

```text
c100090000006500170b0000
```

## 透明传输模式

后续 LoRa 收发需要切回透明传输模式。切换前必须确认当前模块接线和电平，不要只按理论 RX/TX 交叉关系重插线。

## K7 UART 电平注意

KICKPI 文档提示 K7 扩展 UART 默认可能是 1.8V 电平。当前这块板已经实测 E22 相关引脚约 3.2V，并且 UART3 与 E22 读配置成功。后续如果更换板卡或重新接线，需要重新确认电平转换和接线。
